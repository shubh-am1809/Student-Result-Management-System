Student Result Management System (Flask)

Overview
This project is a Student Result Management System built using Python Flask and SQLite. 
It allows administrators and teachers to manage students and
academic results, while students can securely view and download their own results.

The system includes role-based authentication CRUD operations and
PDF result export making it suitable for academic institutions and
internship-level backend projects.

 Features
 Authentication & Roles
- Admin login
- Teacher login
- Student login (Roll No + Date of Birth)
- Role-based access control

Student Management (Admin)
- Add new students
- View student details
- Secure student authentication

Result Management (Admin / Teacher)
- Add student results
- Edit results
- Delete results
- Search results by roll number

Student Dashboard
- View subject-wise marks
- See total marks and percentage
- Download results as PDF

 PDF Export
- Generate professional result report PDF
- Students can download only their own results
- Staff can download any student result

Technologies Used
- Python 3
- Flask– Web framework
- SQLite – Database
- ReportLab – PDF generation
- HTML – Templates
